# 小说协作写作 — 操作指南

## 项目简介

**类型**：日式幻想 · 异世界转生轻小说
**基调**：成长冒险 — 有深度的角色成长，剧情有起伏但不虐，轻松与严肃平衡
**互动方式**：选择分支制 — 每章提供 2-3 个方向，由用户选择推进

## 目录结构

```
exqwl-0o/
├── 操作指南.md       → 本文件，新会话启动时先读这个
├── 设定/             → 世界观规则、魔法体系、种族、角色档案
├── 大纲/             → 整体结构和分章规划
├── 章节/             → 每章一个文件夹，内含正文和笔记
├── 素材/             → 灵感片段、参考资料、未用的点子
└── 草稿/             → 未定稿的片段
```

## 新会话启动流程

每次开新会话，对 Claude 说：

> **"读 exqwl-0o/操作指南.md，按里面的流程继续。"**

Claude 会按以下步骤恢复上下文：

1. 读 `操作指南.md`（本文件）
2. 读 `设定/` 下的所有文件 — 恢复世界观和角色设定
3. 读 `大纲/进度.md` — 了解写到哪了、下一步是什么
4. 读 `大纲/章节摘要.md` — 通过摘要衔接已完成章节的风格、角色、伏笔
5. 读最近一章的正文 — 接上叙事节奏
6. 告诉你当前进度，等你选择：继续写下一章 / 修改已完成章节 / 补充设定

## 写作阶段

### 阶段一：设定搭建 ← 当前
- [ ] 世界观设定（异世界的基本结构、历史背景）
- [ ] 魔法/力量体系
- [ ] 种族与势力分布
- [ ] 主角人设（前世身份、转生方式、初始能力）
- [ ] 核心配角初稿
- [ ] 确定核心主题和风格基调

### 阶段二：大纲规划
- [ ] 写整体结构（预计章节数、主线走向）
- [ ] 写分章大纲（每章的场景、核心冲突、选择分支方向）
- [ ] 确定章与章之间的衔接点

### 阶段三：逐章生成
每章的流程：
1. Claude 先简述本章的预设场景和选择点（基于大纲）
2. 你做选择，Claude 即兴推进叙事
3. 完成后 Claude 将本章正文存入 `章节/第XX章-标题/正文.md`
4. 同步更新 `大纲/进度.md` 中的状态
5. 同步更新 `大纲/章节摘要.md`，补充本章摘要

### 阶段四：串联润色
- [ ] 通读所有章节，调整衔接
- [ ] 回收伏笔，统一风格
- [ ] 补充遗漏的细节描写

## 文件命名约定

- 设定文档：`设定/XXX.md`（如 `世界观.md`、`魔法体系.md`、`角色.md`）
- 大纲文档：`大纲/整体大纲.md`、`大纲/进度.md`、`大纲/章节摘要.md`
- 章节正文：`章节/第XX章-标题/正文.md`
- 章节笔记：`章节/第XX章-标题/笔记.md`
- 素材：`素材/XXX.md`

## 写作原则

1. **互动优先**：每章保留选择分支的感觉，保留"被选择"的张力
2. **跨章呼应**：每章至少埋一个与其他章的连接点
3. **规则一致**：世界观和力量体系必须与设定文档一致，新设定必须先更新设定文件再使用
4. **日式轻小说风格**：
   - **第三人称为主**（贴近主角的限知视角），偶尔切入第一人称内心独白保留吐槽感（以破折号引出或独立成段）
   - 第1-2章为第一人称（序章风格保留），第3章起切换为第三人称
   - 对话活泼，内心戏丰富但不啰嗦
   - 适度使用轻小说特有的吐槽和幽默节奏
   - 战斗场景节奏明快，不拖沓
   - 角色有鲜明的"标签感"，但不扁平
